GTReZ - Gujarati Text Recognition using Zone Segmentation

About the Project

GTReZ focuses on preserving Gujarati manuscripts through digitization. This project presents a robust system for recognizing and converting Gujarati text into an editable digital format. The proposed solution addresses the challenges of recognizing complex scripts and modifiers unique to the Gujarati language.

The system uses advanced methods like zone segmentation and deep learning models to achieve high accuracy and efficiency.

Methodology

Preprocessing

Input images are segmented into upper, middle, and lower zones to reduce the complexity of recognition. Techniques like Intersection over Union (IoU) and center-distance methods are employed to align modifiers correctly with root characters.

Training

The system is trained using YOLOv5 on a dataset of 15,000 images generated by TRDG (Text Recognition Data Generator). Training involved 300 epochs with an optimized learning rate to achieve maximum accuracy.

Post-processing

Recognized characters are combined with their respective modifiers using advanced methods to form complete words and sentences. This post-processing ensures precise text output.

Future Work

The current system focuses on typed Gujarati scripts. Future enhancements include:

  • Extending support to handwritten Gujarati scripts.
  • Expanding recognition to other Gujarati fonts like GJ TT Avantika and LMG Arun.
  • Optimizing the model for better resource utilization and faster inference.
  • Developing mobile and web applications for broader accessibility.
